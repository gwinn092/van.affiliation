## Core Thesis

Recharge speed often matters more than raw watt-hours.

Climate changes system performance dramatically.

Margin reduces stress.

Scale gradually — but design systems that can grow without requiring full replacement.

Avoid replacement cycles. Build upgrade paths.

------------------------------------------------------------

## Tier Structure

Tier 1 — Entry Portable Systems (Under $500)
- 700Wh minimum recommended
- 500Wh is validation tier only
- Works for 300–700Wh/day users
- Requires realistic recharge expectations

Tier 2 — Mid-Tier / Strong Recharge
- 700–1000Wh modern LiFePO4
- Fast AC + alternator charging preferred
- Best for remote workers with moderate daily load

Tier 3 — Margin / Full-Time Stability
- 1500–2000Wh+
- Hybrid solar + alternator strategy
- Built for 800–1000Wh daily usage
- Reduces winter and cloud stress
