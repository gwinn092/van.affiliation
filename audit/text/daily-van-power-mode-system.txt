Choose a mode, follow the rules, and protect tomorrow’s margin. Mantra: pick a mode early, avoid the spiral. Why modes work (recovery-first logic) # Modes reduce decision fatigue and prevent the slow spiral. Instead of reacting to stress late, you adjust early based on recovery reality. When weather shifts or refill windows shrink, modes help you protect baseline loads and cut avoidable drain before it compounds. Related: How to Survive a Cloudy Week in Van Life Normal Mode (default when recovery is strong) # Normal Mode means your refill windows are reliable and your margin is stable. Allowed behavior: DC-first as the default lane AC in planned windows, not background convenience Baseline loads always protected Quick rules (keep it simple): Keep DC-first for routine tasks. Use AC intentionally, then switch back. Do morning and overnight checks. Keep your next recovery window visible. Avoid adding optional loads late. Conservation Mode (when recovery is uncertain) # Conservation Mode is for uncertain refill conditions. Common triggers: Cloud cover or weak sun Shade-heavy parking No driving recovery planned Heat stacking baseline demand Allowed behavior: Baseline-first, comfort-second Tight AC discipline (less “convenience by default”) Early adjustment instead of late panic Quick rules (tighten early): Protect fridge, ventilation, and essentials first. Trim optional loads immediately. Keep inverter usage intentional and brief. Put recovery planning at the top of the day. Re-check mode before evening. Recovery Mode (when you’re behind) # Recovery Mode is the reset state. Use it when margin is already slipping or after multiple weak refill windows. Common triggers: You wake up behind Multiple weak-weather days stack Baseline feels unstable by midday Allowed behavior: Plug in early if available Use driving recovery when practical Baseline-only until stability returns Quick rules (reset first): Recover first, spend later. Keep optional loads off. Use shore as anchor when possible. Layer driving/solar support where practical. Exit Recovery Mode only after baseline stability returns. Read next: Shore Power Recovery Routine for Van Life Solar vs Alternator Charging Mode switching: the simple decision tree # IF baseline feels stable and refill windows are reliable, THEN stay in Normal Mode . IF weather weakens or refill certainty drops, THEN switch to Conservation Mode early. IF you’re already behind, THEN move to Recovery Mode immediately. IF you can plug in today, THEN prioritize a Recovery Mode reset first. IF you’ll drive today, THEN treat it as recovery and keep optional loads low. IF neither plug-in nor driving is available, THEN stay in Conservation Mode and protect baseline. IF evening margin looks unstable, THEN tighten one mode level before bed. IF morning starts tight, THEN stay in Recovery Mode until baseline stability returns. Night protects tomorrow. FAQ # How do I know when to switch modes? # Switch based on recovery certainty, not mood. If refill windows are weak or uncertain, tighten early. What if my partner wants comfort mode always? # Agree on a shared baseline-first rule and set comfort windows intentionally. A clear mode system reduces conflict and surprise stress. Does this replace a bigger battery? # No. Modes improve workflow. Bigger buffer can still help, but better routine often improves stability first. Can I stay in Normal Mode all week? # Only if recovery stays reliable. If weather or refill changes, switch before you fall behind. What is the biggest mode mistake? # Waiting too long to switch. How does this connect to inverter habits? # Mode changes should include inverter behavior changes, especially moving from Normal to Conservation or Recovery. Is this only for small systems? # No. Larger systems benefit too, especially when conditions become unpredictable. Where should beginners start? # Start with Normal Mode rules plus morning and overnight checklists, then practice switching earlier than you think you need to.
