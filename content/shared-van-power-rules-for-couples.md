# Shared Van Power Rules for Couples: A Simple Agreement That Prevents Daily Stress

Most van power fights are recovery fights in disguise. When margin is unclear, small decisions feel personal fast. A shared routine turns daily tension into simple agreements.

**Mantra:** protect baseline first, then negotiate comfort.

## The shared goal: baseline stability first

Agree on one priority before anything else: protect baseline stability.

Baseline usually means:
- Fridge
- Essential ventilation
- Essential devices

Comfort comes after baseline. This keeps stress lower when weather or recovery windows get messy.

## The 3-mode agreement (Normal / Conservation / Recovery)

Use one shared mode each day.

Related system:
- [Daily Van Power Mode System](/daily-van-power-mode-system/)

### Normal Mode
- Baseline loads are protected by default.
- Comfort loads are allowed in planned windows.
- AC use stays intentional, not automatic.
- Both people follow morning and overnight checks.

### Conservation Mode
- Baseline loads stay protected; comfort loads tighten.
- Optional AC use is reduced and planned.
- Cooking decisions follow recovery reality, not routine habit.
- Refill windows become top priority.

### Recovery Mode
- Reset first, comfort second.
- Baseline-only behavior until stability returns.
- Plug-in or driving recovery windows are prioritized.
- No surprise load stacking.

## The daily meeting (2 minutes, morning + night)

Run a short check-in twice a day:
- **Morning:** decide mode and today’s recovery path.
- **Night:** protect overnight margin and set tomorrow’s plan.

Use this script:
- “Today we’re in **___** mode.”
- “Today’s recovery window is **___**.”
- “Today’s comfort windows are **___**.”

**Tiebreaker rule:** when you disagree, choose the tighter mode for a few hours and re-check at midday.

## Comfort windows (how to say yes without regret)

Comfort does not need to disappear. It needs timing.

Set planned windows for:
- Cooking
- Laptop-heavy blocks
- Extra fan use
- Entertainment loads

If recovery is strong, comfort windows expand. If recovery is weak, comfort windows tighten.

## The top 7 arguments (and the rule that solves each)

### Fans vs battery
**Rule:** protect essential ventilation; optional fan use is mode-based.  
**Why:** sleep and safety stay stable without pretending every fan choice is equal.

### Cooking tonight vs tomorrow margin
**Rule:** cook on recovery days, conserve on uncertain days.  
**Why:** cooking choices are recovery choices, not just meal choices.

### One partner leaves inverter on
**Rule:** default OFF unless AC is actively needed.  
**Why:** it reduces passive drain and avoids avoidable overnight stress.

### “Just charge everything”
**Rule:** charge baseline devices first, then optional items.  
**Why:** priority order prevents late-day surprises.

### Parking shade vs sun
**Rule:** choose parking based on mode, not preference alone.  
**Why:** recovery needs can outweigh convenience on weak-weather days.

### Driving vs staying put
**Rule:** choose the option that best supports today’s recovery plan.  
**Why:** movement decisions should serve stability, not habit.

### Surprise AC stacking
**Rule:** no new optional AC tasks without a quick mode check.  
**Why:** stacking is where stable days turn into stressed days.

## FAQ

### What if one person works remotely and needs laptop time?
Define work windows in advance and protect them in mode planning. Shift optional comfort loads around those blocks instead of competing with them.

### What if one person runs hot and needs fans?
Treat essential ventilation as baseline, not optional comfort. Adjust other loads first when you need to protect margin.

### How do we stop blaming each other for low battery?
Use shared mode rules and short check-ins. Let the system carry decisions so people don’t need to argue each time.

### What if we disagree on mode?
Use recovery reality as the tiebreaker: weather, refill windows, and baseline protection decide mode. If it’s unclear, choose the tighter mode briefly and re-check.

### Do we need to discuss this every day?
Short daily check-ins prevent larger arguments later. Two minutes now saves stress later.

### Can this work for roommates or families too?
Yes. The same baseline/comfort and mode rules work for any shared setup.

### What if we only plug in occasionally?
Then recovery windows become more important, not less. Plan comfort windows around known refill opportunities.

### Is this only for small systems?
No. Larger systems reduce urgency, but shared routines still prevent avoidable stress.
